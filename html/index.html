Portfolio

Chopsticks Animated

Using techniques from my Computer Animation and Artificial Intelligence classes, I made a single-player virtualization of the game "Chopsticks".

Speed-Viewer

Speed-Viewer is an interactive data visualization webapp that displays speedrun data from Speedrun.com. It utilizes the d3.js library to plot scatterplots of games added by the user, and is styled using Tailwind CSS. In this group project, I was responsible for laying a lot of the functional groundwork: I created the initial design for the interface and implemented much of the graphing functionality.

Accordion Solitaire (WIP)

This is a small passion project of mine that virtualizes the card game Accordion Solitaire with HTML, CSS, and JS. It is functional, but it still needs some polishing.

PhotoCoach

For my senior capstone project, my team and I designed and implemented PhotoCoach, a mobile application intended to provide users actionable feedback to help them develop thier mobile photography skills. I helped implement and automate testing on the frontend, which was developed using Flutter.

Ray Tracer

My Computer Graphics class required me to implement a ray tracer from scratch using OpenGL and C++. It handles sphere and plane collision, reflections, and lighting with shadows.

Roadies - Breakaway Ministries

For my last three undergraduate semesters, I had the priviledge of serving on Breakaway Ministries' Roadie team.
